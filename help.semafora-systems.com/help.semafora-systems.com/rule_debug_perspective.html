Debugging Rules

  

OntoStudio includes a debugging environment that is oriented towards an intuitive process of debugging. If you have a query that is not returning the expected results, you can use the debugger to analyze the problem. It helps you to identify whether rules are not connected properly or to isolate the rule(s) that is (are) not returning the correct results.

1.To debug a query, make a right mouse click on the selected query and then click on Debug query.
