Connector.Cache.Lifetime

query,unlimited.

query - tuple cache is removed after the query execution. This is the default value for this option.

unlimited - tuple cache is not removed after the query execution and can be used for the next query. This option is better for performance and should always be used if a database won't be updated frequently. Cache update should be done manually (see "Clear connector cache command").

The option affects all connectors.
